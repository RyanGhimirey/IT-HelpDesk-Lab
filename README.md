# IT Help Desk Simulation Lab

This repository contains a selection of help desk tickets completed as part of a hands-on IT support lab. Each example reflects common real-world scenarios and focuses on practical troubleshooting, system configuration, and user support.

## Scope of Work
- Active Directory user management and access issues  
- Printer setup and troubleshooting  
- VPN connectivity and remote access support  
- Network drive mapping and permissions  
- Remote desktop troubleshooting  

## Ticket Format
Each ticket includes a brief description of the issue, the steps taken to troubleshoot and resolve it, and the final outcome. Screenshots are included where relevant to support the process.

## Additional Context
Over 100 support scenarios were completed as part of this lab. This repository highlights a selection of those cases to demonstrate range and consistency.
